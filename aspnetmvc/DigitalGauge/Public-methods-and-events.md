## Methods


### destroy()


To `destroy` the digital gauge


{% highlight CSHTML %}
 
@(Html.EJ().DigitalGauge("Gauge1"))

{% endhighlight %}

{% highlight js %}
 
var gauge = $("#Gauge1").data("ejDigitalGauge");
gauge.destroy();
   
{% endhighlight %}
