## Methods

### navigateTo(latitude, longitude, level)

Method for navigating to specific shape based on latitude, longitude and zoom level, you can use `navigateTo` method.


{% highlight CSHTML %}
 
@(Html.EJ().Map("Map1"))

{% endhighlight %}

{% highlight js %}
 
var map = $("#Map1").data("ejMap");
map.navigateTo();
   
{% endhighlight %}


### pan(direction)

{% endhighlight %}
