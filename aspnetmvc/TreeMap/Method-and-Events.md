# Methods and Events

## Method

## Refresh()

`refresh()` method is used to reload the treemap with updated values.

#### Returns: void
{% highlight CSHTML %}

    @(Html.EJ().TreeMap("treemap"))

    <script type="text/javascript">
        $("#treemap").ejTreeMap();
        var treemapObj = $("#treemap").data("ejTreeMap");
        treemapObj.refresh();
    </script>

{% endhighlight %}
